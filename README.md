# cat-brush-game
A web game where the goal is to groom a cat using the mouse, with planned authentication and leaderboard.
A small educational project for practicing TypeScript and frameworks.

## Stack
- Frontend: Next.js, TypeScript, Canvas API
- Backend: Next.js API routes, Prisma, PostgreSQL
- Auth: Auth.js
- Tests: Playwright (E2E + API), Vitest (unit)

## Target
Improve TypeScript skills, gain an understanding of backend and frontend architecture, learn how to use database migrations, and practice building automated tests from scratch
